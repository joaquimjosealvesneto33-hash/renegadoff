Se você quer colocar sua página no ar de graça e aparecer no Google, siga este passo a passo: 1. **Criar conta no GitHub** — Acesse [https://github.com](https://github.com) e crie sua conta. 2. **Criar um repositório** — Depois de logado, clique em *New Repository*, dê um nome como `renegadoff` e deixe público. 3. **Enviar seus arquivos** — Faça upload deste arquivo HTML e suas imagens para o repositório. 4. **Ativar GitHub Pages** — Vá em *Settings* → *Pages* → selecione a branch principal e a pasta `/root` (ou `/docs`) → Salve. 5. **Acessar seu site** — O GitHub vai gerar um link do tipo `https://seuusuario.github.io/renegadoff`. 6. **Enviar para o Google** — Acesse [Google Search Console](https://search.google.com/search-console/) → *Adicionar propriedade* → cole o link do GitHub Pages → solicite indexação. Isso tudo é 100% gratuito e o site ficará online para qualquer pessoa pesquisar.
